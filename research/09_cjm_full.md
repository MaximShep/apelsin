# CJM: карта «Апельсин» — путь пользователя

> Все факты — с источниками. Поведенческие детали («мысли», «настроение») — авторская интерпретация на основе отзывов.
> Актуально: апрель 2026. Карта — Апельсиновая карта Альфа-Банка, система МИР.

---

## Контекст: кто этот человек

Маша, 21 год. Живёт одна или с соседями. Есть карта Т-Банка (основная), иногда Сбер. В Пятёрочку ходит 2–3 раза в неделю. Сканирует QR X5 Клуб на кассе, но воспринимает как «просто скидки» — не думает об этом.

## ЭТАП 2: ИЗУЧЕНИЕ И РЕШЕНИЕ

**Что происходит:**
Гуглит «карта апельсин x5», открывает 2–3 ссылки: myapelsincard.ru, banki.ru, статью в Дзене.

| | |
|---|---|
| **Цель** | Понять — стоит ли |
| **Что видит на myapelsincard.ru** | «7% в Пятёрочке и Перекрёстке», «бесплатно навсегда», «виртуальная карта за 15 секунд». |
| **Что замечает при внимательном чтении** | 7% действуют **только на товары без акции** (белые ценники). На жёлтых ценниках с акцией — нет. [hranidengi.com, авг 2024] |
| **Мысли** | *«Подождите. Я беру акционные товары. Значит 7% почти не будет?»* — и это убивает половину ценностного предложения. |
| **Что читает в отзывах** | «Подключили Альфа-Смарт 399 руб/мес без предупреждения, с января по май списали 1600 рублей — весь кешбэк поглотила комиссия» [banki.ru #12287879, май 2025]. «Пожалела, что открыла — обязательно спросите менеджера про скрытые подписки» [irecommend.ru, заголовок]. |
| **Решающий фактор «за»** | Бесплатное обслуживание навсегда. Нет риска потерять деньги, если не понравится. |
| **Pain** | Механика сложнее, чем кажется: 6% карта + 1% X5 Клуб = 7% только на неакционные товары. Пользователь [banki.ru #12144560, фев 2025]: «Не понимаю, почему рекламируют 7%, когда на деле 6% — один процент где-то спрятан, сотрудники банка и магазина говорят разное». |
| **Настроение** | 🤔→😟 От любопытства к тревоге из-за скрытых подписок. |

## ЭТАП 3: ОФОРМЛЕНИЕ

**Ключевой факт:** Офис не нужен. Оформление только онлайн — через myapelsincard.ru или alfabank.ru. В отделениях Альфа-Банка карта **не выдаётся**. [hranidengi.com, форум]

**Выбор типа карты — первое решение:**

| | Виртуальная | Пластиковая |
|---|---|---|
| Скорость | 15 секунд | 1–2 дня курьером |
| 7% в X5 | Только 2 месяца, потом 1% | Постоянно |
| Источник | [myapelsincard.ru], [alfabank.ru] | [bankprofi.ru] |

*Большинство молодёжи выбирает виртуальную — быстро. Но ловушка: через 2 месяца остаётся 1% вне X5, и карта «умирает».*

**Заполнение анкеты:** ФИО, дата рождения, телефон, email, адрес доставки. Согласие с документами — чекбокс.

| | |
|---|---|
| **Pain** | При онлайн-оформлении курьер при доставке может подключить платную подписку «Альфа-Смарт» (399 ₽/мес) — клиент думает, что просто подписывает получение карты. [banki.ru #12287879] |
| **Мысли** | *«Подписал получение карты. Или я что-то ещё подписал?»* — неясно в момент. |
| **Настроение** | 🙂 При заказе — ок. Тревога появится позже, когда обнаружит списание. |

## ЭТАП 4: ПОЛУЧЕНИЕ ПЛАСТИКОВОЙ КАРТЫ

| | |
|---|---|
| **Срок** | 1–2 рабочих дня [bankprofi.ru] |
| **Что происходит** | Курьер звонит, согласовывает время. Приносит конверт с картой и документами. Маша подписывает получение. |
| **Что в конверте** | Пластиковая карта. Документы с условиями (мелкий шрифт). |
| **Первое впечатление** | Стандартный конверт Альфа-Банка. Нет wow-эффекта — не такой, как металлическая карта Т-Банка или красивая упаковка. |
| **Pain** | Курьер может попросить подписать подключение «Альфа-Смарт» в пакете документов — если не читать внимательно, не заметишь. [banki.ru #12287879] |
| **Настроение** | 😐 Нейтральное. Карта пришла — окей. |

**После получения — установка PIN:**
Через приложение Альфа-Банка или по звонку 8 800 505 2000. [myapelsincard.ru/common-info/dc/]

**Привязка к X5 Клубу — автоматически или нет:**
Если номер телефона в Альфа-Банке совпадает с номером в X5 Клубе — привязка автоматическая, старые баллы сохраняются. Если не совпадает — нужно связать вручную. [x5club.ru/faq/bankCardX5Club]

| | |
|---|---|
| **Pain** | Маша могла регистрировать X5 Клуб на другой номер — тогда придётся разбираться с привязкой. Неочевидно. |
| **Настроение** | 😐→😕 Нейтральное или лёгкая путаница. |

## ЭТАП 5: ПЕРВАЯ ПОКУПКА

### На кассе в Пятёрочке

**Механика (это важно и неочевидно):** Чтобы получить 7%, нужно сделать ДВА действия:
1. **Сканировать штрихкод с пластиковой карты** (или QR из приложения Апельсин)
2. **Оплатить картой Апельсин**

Только при таком сочетании начисляются 7%. Оплата без сканирования — нет кешбэка. [x5club.ru/faq/bankCardX5Club]

| | |
|---|---|
| **Мысли на кассе** | *«Так, штрихкод сначала или карта? Кассир говорит "всё равно". А приложение говорит сначала штрихкод»* |
| **Pain** | Не все кассиры знают правильный порядок. Нет подсказки на экране кассы. При Alfa Pay / Mir Pay баллы начисляются с задержкой до 14 дней. [banki.ru #11762652, окт 2024] |
| **Настроение** | 😰 Тревога — не хочется облажаться и потерять кешбэк. |

### После оплаты

| | |
|---|---|
| **Что происходит** | При оплате физической картой + штрихкод — баллы зачисляются мгновенно. [x5club.ru/faq/orange] |
| **Push-уведомление** | «+N баллов X5 Клуб» (с сентября 2025 — баллы, не апельсинки) [x5club.ru/faq/orange] |
| **Мысли** | *«Сколько это рублей? Надо посчитать — 10 баллов = 1 рубль... значит 350 баллов = 35 рублей? Хм»* |
| **Pain** | Конкуренты (Т-Банк, Сбер) пишут «сэкономили 35 ₽» сразу. Апельсин пишет «+350 баллов» — мозг не связывает это с деньгами автоматически. |
| **Настроение** | 😐 Нейтральное. Ожидали больше радости. |

### Открытие про акционные товары

| | |
|---|---|
| **Момент** | Смотрит в историю начислений — кешбэк меньше, чем ожидала |
| **Что узнаёт** | 7% начисляются **только на товары без акции** (белые ценники). Акционные товары (жёлтые ценники) — исключены. [hranidengi.com форум, авг 2024] |
| **Мысли** | *«Я специально беру акционное. Выходит, 7% у меня почти не будет»* |
| **Pain** | Это не написано крупно нигде при оформлении. Обнаруживается только на практике. |
| **Настроение** | 😤 Разочарование. Ощущение скрытого условия. |

### Первая попытка потратить баллы

| | |
|---|---|
| **Что узнаёт** | С физической картой (штрихкод) — можно списать только **50% от суммы чека**. С QR из приложения Апельсин — **100%**. [banki.ru #12235923, апр 2025] |
| **Мысли** | *«Почему только половину? Нигде не было написано»* |
| **Pain** | Отдельное приложение Апельсин нужно для полноценного использования — но оно нестабильно. App Store: 2.5/5 (147 оценок). Типичный отзыв: «Не грузит, выдаёт ошибку при авторизации» [App Store, янв 2025]. «Не подвязывается к Пятёрочке» [RuStore, апр 2026]. |
| **Настроение** | 😤 Ещё одно скрытое ограничение. |
