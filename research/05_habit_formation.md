# Психология формирования привычки в финтехе

## Ключевые данные

| Метрика | Значение |
|---|---|
| Доля выбора «основной карты» в первые 90 дней | >70% |
| D30 retention fintech (лидеры рынка) | 73% |
| D30 retention fintech (средний рынок) | 30% |
| D90 retention «хороший показатель» | 10–30% |
| Рост вовлечённости при переменных наградах vs обычный банкинг | +35% за 90 дней |
| Эффект мгновенного уведомления о кешбэке vs еженедельной сводки | ×3 эффективнее |
| Push-уведомления vs без них | ×4 вовлечённость (при умеренном использовании) |
| Социальное доказательство vs рациональный выбор | Пиры выигрывают у >60% молодёжи |

## Модель Hook (Нир Эяль) применительно к Апельсину

```
ТРИГГЕР
Внешний: «ты в Пятёрочке» / push «сегодня заработаешь 80 руб»
Внутренний: привычка «иду за продуктами»
        ↓
ДЕЙСТВИЕ (минимальная фрикция)
Достать карту / телефон → приложить → оплатить
        ↓
ПЕРЕМЕННАЯ НАГРАДА
Уведомление: «+80 апельсинок» — сумма каждый раз разная
Сюрприз-бонусы от партнёров
Прогресс к следующему уровню / цели
        ↓
ИНВЕСТИЦИЯ (то, что создаёт возврат)
Накопленные баллы (жалко терять)
Выбранные категории кешбэка (время потрачено)
Приглашённые друзья (социальный граф)
Уровень в программе (статус)
```

## Что делает карту «основной»: 5 механизмов

### 1. Технический лок-ин
Карта по умолчанию в Apple Pay / Google Pay / SberPay. Первый кошелёк = первый выбор. Борьба за «позицию по умолчанию» — критична.

### 2. Совпадение с топ-категорией расходов
Карта с максимальным кешбэком там, где человек тратит больше всего → рефлекторный выбор. У молодёжи топ-категории: продукты, кафе, доставка еды, транспорт.

### 3. Социальный лок-ин
Друзья пользуются → social proof → переключение. Особенно сильно для 18–25. Одного человека в компании достаточно, чтобы «заразить» группу.

### 4. Streak / прогресс / что теряешь
Накопленный статус, уровень, серия активных недель — потеря воспринимается острее, чем выгода от нового (теория перспектив Канемана). «Не хочу терять прогресс» удерживает сильнее, чем «хочу получить бонус».

### 5. Немедленное вознаграждение
**Критически важно:** кешбэк должен быть виден сразу. Не «накопится к концу месяца», а push-уведомление через 3 секунды после оплаты. Это и есть «дофаминовый удар», который закрепляет нейронный паттерн.

## Окно возможностей: когда ловить молодёжь

| Жизненное событие | Почему это момент |
|---|---|
| 18 лет / первая своя карта | Нет сформированных привычек |
| Поступление в вуз / переезд | Новый паттерн жизни и трат |
| Первая работа / стажировка | Первые «взрослые» деньги |
| Переезд в другой район / город | Новый «домашний» магазин |

**Вывод:** Апельсину нужно присутствовать в этих моментах — через кампусные программы, онлайн-каналы, реферальные механики.

## Барьер инерции: почему 7% не работают сами по себе

Переключение карты требует усилия → мозг его избегает. Даже если Апельсин выгоднее, человек продолжает платить привычной картой. Рациональная выгода не побеждает поведенческую инерцию.

**Решение:** Снизить усилие переключения до нуля + дать немедленную ощутимую выгоду прямо в момент первого использования.
